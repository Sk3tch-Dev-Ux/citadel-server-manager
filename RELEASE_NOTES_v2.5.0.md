## Live Ops Dashboard

Five new admin-grade systems, all included in your existing subscription.

### Kill Feed + PvP Analytics
Real-time PvP feed, per-wipe leaderboard, analytics tab with top weapons, distance histograms, and top-3 killer comparison.

### Chat Log Viewer
Searchable chat archive with channel badges, player filter, message contains, and CSV export. Live socket feed with pause/resume.

### Player Profiles & History
Persistent cross-wipe profiles: session history, lifetime K/D, alias tracking, chat log, and an admin notes timeline.

### Automated Backup Scheduling
Dedicated Backups page with frequency presets (15min → 24h), retention by days, one-click restore, archive preview.

### Admin Power Tools
Audit log, outbound webhooks, priority queue, system dashboard.
