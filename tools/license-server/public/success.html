Welcome to Citadel!

Your purchase is complete. Check your email for your license key and repository access.

📦 Step 1 — Get the Code

  1. Check your GitHub notifications for a repository invitation
  2. Accept the invite to the private Citadel repo
  3. Clone the repository to your server

🔑 Step 2 — Activate Your License

  1. Check your email inbox (and spam folder) for a message from Citadel Licenses
  2. Follow the setup guide in the repository README
  3. Open your Citadel dashboard → Settings → License
  4. Paste the license key from the email and click Activate
