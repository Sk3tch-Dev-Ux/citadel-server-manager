Citadel service isn't responding

The Citadel Windows service may have crashed or failed to start.

What to try:

  1. Open Services (services.msc)
  2. Find Citadel DayZ Server Controller
  3. Right-click → Start
  4. If it won't start, check C:\Citadel\data\service.log
